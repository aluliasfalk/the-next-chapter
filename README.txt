THE NEXT CHAPTER — VERSION 6.0

WHY THIS IS THE PERMANENT EDITION
- One reliable app page; Timeline, Travel, Story, Memories and Countdown cannot create 404 page errors.
- Photos are separate normal files for faster loading.
- Every file is in the repository root—there are no nested folders to miss during upload.
- Wedding date is October 21, 2000 everywhere.
- Wedding photos are framed lower to preserve the full tops of your heads.
- Optional original ambient music is included and off by default.
- Installable on iPhone, iPad, Android and desktop.
- Offline support and automatic cache updates.

UPLOAD INSTRUCTIONS
1. Unzip this package.
2. In your GitHub repository, delete the old files and old assets folder.
3. Upload EVERY file from this package into the main/root repository area.
4. Commit the changes.
5. Wait 2–5 minutes.
6. Open:
   https://aluliasfalk.github.io/the-next-chapter/
